Cass finds Alex in the market on Meridian Street — or seems to. Claims to have
been VOSS's contact on the west side. Knows the seal. Knows the name on the
destination. Offers a safer route.

Alex takes the offer because the alternative is worse.

---
**Craft note — Subplot / Ally Introduction:** The audience should not know if
Cass is trustworthy. Neither should Alex. This ambiguity is the subplot. It runs
parallel to the main plot question (will Alex deliver?) asking a second question
(who is Cass?). A good B-plot mirrors the A-plot thematically: the main plot
is about delivering something despite not knowing if it will matter; the subplot
is about trusting someone despite not knowing if they deserve it.
